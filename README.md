# Bookmarklets
Original plan is to create Bookmarklets that will store a Job posting information locally for your job hunting reference.

I have been job hunting for over a year with a lot of ghost responses. So I have established standards that will help me capture information and be able to reference it at a future date.

## Long-term Goals
Extract information locally as the Job Posting often expires overtime. Capturing the information allows for ability to run AI on the extracted information from various postings that you have been interested in. Running all types of analytics and keyword usage across all files.

1. Create a .TXT file to be stored locally.
    - filename format: YYYYMMDD - Company Name__Job_Title.txt
    - example: 20260304 - S&P Global__Engineering.txt
        - YYYYMMDD is the current date
2. Extract information into Google Sheets to help track Application submission and Response information
